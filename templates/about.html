About Trip Planner App

Project Description

Trip Planner App is a web application that helps travelers explore and discover various places of interest in different cities. Using TripAdvisor's extensive database, this application provides information about restaurants, hotels, and attractions in selected cities.

Users can easily navigate between different cities and categories, view photos, addresses, and visit the official websites of these establishments. The application is designed to be user-friendly and visually appealing, making travel planning a breeze.

Features

  • Browse restaurants, hotels, and attractions in different cities
  • View photos and details of each place
  • Get direct links to official websites
  • Intuitive carousel navigation for easy browsing
